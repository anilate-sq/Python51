# Начало работы с БД на Python

Работа начаинается с подбора нужной библиотеки

Список ключевых:
- **psycopg2** - Библиотека для работы с PostgreSQL;
- **sqlite3** - Библиотека для работы с SQLite;
- **mysql-connector-python** - Библиотека для работы с MySQL;
- **mysqlclient / PyMySQL** - Альтернативный вариант для работы с MySQL;
- **pymssql** - Библиотека для работы с MSSQL.

**psycopg2** - самый популярный адаптер для работы с PostgreSQL в Python.
