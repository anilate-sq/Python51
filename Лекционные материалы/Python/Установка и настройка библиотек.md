# Установка и настройка

## Основная библиотека

```bash
pip install psycopg2-binary
```

## Работа с базой
```bash
createdb -U postgres dbname # Создание базы данных
psql -U postgres -d db_name -f db_script.sql # Выполнение скрипта
```

> **P.s**: на практике лучше работать вручную и в полноценных проектах лучше брать `psycopg`(без `-binary`)
