# REST API

**Application Programming Interface(API)** — инструмент, позволяющий разным приложениям обмениваться данными и функциями.

**REST API** - подход к построению API через HTTP-запросы

**Ключевые принципы:**
- Использет HTTP(GET, POST, PUT, PATCH, DELETE)

**Логика работы:**

```bash
GET /movies # Получение списка фильмов
POST /movies # Добавление нового фильма
GET /movies/1 # Получение конкретного фильма
```

**FAST API** - это python-фрейморк направленый на разработку WEB-приложений и API

**Pydantic** - это библиотека предназначенная для проверки и преобразования данных, если сказать по простому, то это автоматическая валидация
