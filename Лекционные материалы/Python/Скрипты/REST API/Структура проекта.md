# Структура проекта

```
REST API/
|
|— main.py(Точка входа в программу)
|— database.py(Создание подключения к БД)
|— dependencies.py(Зависимости)
|— models.py(Модели, как правило делаются в отдельной папке)
|— schemas.py(Внешние модели, как правило делаются в отдельной папке)
|
|— routers/(Роутеры) 
|   |— movies.py (Маршруты /movies)
|   |— reviews.py (Маршруты /reviews)
|
```
