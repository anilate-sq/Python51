# Терминологический словарь

**Model** - описание таблицы в базе данных, внутренняя модель

**Schema(Pydantic)** - структура данных для API, внешняя модель

**Router** - это группа endtoint-ов

**Endpoint** - это конкретный URL-адрес с методом(GET, POST и т.д.)

**Dependencies** - механизм внедрения зависимостей(База данных является сторонним сервисом)

**Слои** - это способ разделения логики и ответственности в коде
