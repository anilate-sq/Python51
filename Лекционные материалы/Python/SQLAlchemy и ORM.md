# Введение в ORM

**Object-Relation Mapping(ORM)** — это подход, позволяющий взаимодействовать с базами данных на уровне объектов. Основная фишка заключается в абстрагировании от SQL-запросов, что позволит сделать код более простым и читаемым.

## Преимущества ORM

- Работа с БД через классы и объекты
- Автоматическая генерация SQL-запросов без необходимости писать их вручную
- Упрощение взаимодействия между программой и хранилищем данных
