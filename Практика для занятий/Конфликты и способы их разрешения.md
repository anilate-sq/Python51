# Практическое задание по конфликтам в Git

## Задание
1. Создать новый репозиторий на github;
2. Реализовать в рамках репозитория 2 ветки(main и dev);
3. Спровоцировать конфликт слияния локальных веток;
4. Разрешить конфликт и закомитить изменения;
5. Спровоцировать конфликт слияния локальной и удаленной веток;
6. Разрешить конфликт и зафиксировать изменения.

## Формат сдачи
- Ссылка на git-репозиторий
