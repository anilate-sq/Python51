# Практическое задание по выборке и агрегации данных

**Выборка данных:**
1. Получить информацию о всех книгах в категории 3;
2. Получить информацию о всех книгах вышедших до 1888 года;
3. Получить информацию о всех книгах вышедшив в 21 веке;
4. Получить информацию о книгах в одном из следующих жанров(2, 3, 5), через jenre_id;
5. Вывести всех авторов с незаполненным описанием(description);
6. Вывести список из 3 самых новых книг;
7. Вывести список из 5 самых старых книг;
8. Вывести все книги, в названии которых есть слово "убийство";
9. Вывести все книги, название которых начинается со слова "Повесть".

**Агрегация данных:**
1. Вывести общее количество книг;
2. Вывести количество книг в каждом жанре;
3. Вывести самую низкую цену за книгу в жанре 3;
4. Вывести среднию стоимость книг в каждом жанре;
5. Посчитать суммарную стоимость книг каждого жанра.
